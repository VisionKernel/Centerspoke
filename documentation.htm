VisionKernel CLI Documentation

v1.0.0 LTS - VisionKernel Corp.

visionkernel - available commands:

-v, -V, -version

These tags will output the current installed version of the VisionKernel application.

-help

Thia tag will output supported commands to the console.

update

This command will update the VisionKernel application to the most recent version. Add -LTS to the command to update to the most recent version with long term support.

Run "-a" tag to update all datafeeds and services, "-s" for all services and "-d" for all datafeeds.

install

This command will install datafeeds and services.

"visionkernel install datafeed example" will install the API example, "visionkernel install service example" will install the service example.

startapi

"visionkernel startapi example" will start the API example and start displaying data values until exited.

nitroexport

"visionkernel nitroexport example.txt" will upload the example.txt file directly to your database

To specify database and/or table run "visionkernel nitroexport example.txt -db example_database -t exampletable"

smartexport

"visionkernel smartexport example.txt" will review the example.txt file for irregularities before uploading to the database. (You can configure the database and table same way as nitroexport)

convert

"visionkernel convert file.xls > file.csv" will convert the file.xls file to a file.csv file and save it under root directory

use and on

"visionkernel use exampleservice on file.xls" will use the selected service on specified file

delete

"visionkernel delete service exampleservice" will delete the service exampleservice

"visionkernel delete datafeed exampleapi" will delete the data API exampleapi

print

"visionkernel print installed" will output all installed libraries. "visionkernel print cloudconfig" will output the cloud database(s) info.
